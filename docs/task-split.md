# 开发任务拆分

## 第一阶段：MVP 骨架

- 完成前后端独立工程初始化
- 建立后端分层结构与统一响应
- 建立 Vue 路由、Pinia、权限守卫与基础布局
- 配置 Flyway 初始化核心表和演示数据

## 第二阶段：认证与权限

- 登录、注册、JWT 鉴权
- 角色路由控制
- 学生端与管理员端菜单隔离

## 第三阶段：学生端核心功能

- 院校查询与详情
- 学习计划与学习任务
- 资料上传与下载
- 讨论区发帖与评论
- 模拟测试与自动判分
- 错题本与成绩结果

## 第四阶段：管理员端核心功能

- 仪表盘统计
- 用户管理
- 院校/专业/简章维护
- 资料审核
- 政策资讯维护
- 公告管理
- 系统配置与日志

## 第五阶段：毕业设计完善项

- 增加分页、条件筛选、图表统计
- 完善操作日志埋点
- 细化提醒功能
- 增加接口测试、前端页面细节优化
- 补充论文截图、流程图与部署文档
