# 需求说明

## 项目题目

基于 SpringBoot 的考研助手系统的设计与实现。

## 角色定义

- `STUDENT`：学生用户
- `ADMIN`：管理员用户

## 设计约束

- 数据库表名、字段名、索引名、后端实体类字段、DTO 字段、API 参数全部采用英文命名
- 表名使用小写下划线，Java 字段使用驼峰命名
- 前后端分离，统一 RESTful API 与统一响应结构
- 后端固定技术栈：Spring Boot 3、Spring Security、JWT、JPA、MySQL、Flyway
- 代码分层：`api / application / domain / infrastructure / common`
- 文件上传采用本地存储
- 讨论区仅实现发帖、列表、详情、评论
- 模拟测试仅实现题库答题、自动判分、错题记录、错题解析

## 学生端功能

- 注册登录
- 个人信息与个人中心
- 院校、专业、招生简章、报录比、政策资讯查询
- 学习计划与学习任务
- 资料上传、资料下载、审核状态查看
- 讨论区
- 模拟测试
- 成绩结果
- 错题本

## 管理员端功能

- 用户管理
- 院校维护
- 专业维护
- 招生简章维护
- 资料审核
- 政策资讯维护
- 试卷管理
- 题目管理骨架
- 公告管理
- 系统配置
- 操作日志
- 仪表盘
